# 📋 Clean Task Dashboard - v5.0

Este projeto é a aplicação final do módulo de JavaScript da **Infinity School**. Trata-se de um ecossistema de gerenciamento de tarefas (To-Do List) focado em produtividade, com persistência de dados e monitoramento de tempo.

## 🚀 Funcionalidades Principais

- **Gestão de Tarefas (CRUD):** Adicione, edite, remova e organize suas atividades diárias de forma dinâmica.
- **Persistência Local:** Utiliza a *LocalStorage Engine* para garantir que seus dados permaneçam salvos mesmo após fechar o navegador.
- **Interface Dashboard:** Design moderno, limpo e responsivo, focado na experiência do usuário (UX).
- **Status em Tempo Real:** Badges dinâmicos que indicam se uma tarefa está "Pendente" ou "Concluída".

## ⏱️ Controle de Horários (Diferencial)

Um dos grandes destaques desta versão é o módulo de monitoramento temporal, que oferece maior controle sobre a rotina:

1.  **Prazo Previsto (Deadline):** Ao criar uma tarefa, o usuário pode definir um horário específico para o cumprimento da atividade.
2.  **Registro de Conclusão (Timestamp):** O sistema registra automaticamente o horário exato em que o usuário marcou a tarefa como concluída.
3.  **Comparativo de Performance:** Permite visualizar o horário previsto versus o horário em que a tarefa foi realmente finalizada, facilitando a análise de produtividade.

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3:** Uso de variáveis customizadas, Flexbox e sombras suaves para o layout Clean.
- **JavaScript Moderno (ES6+):** Manipulação avançada do DOM com `createElement` e `appendChild`.
- **Tratamento de Erros:** Implementação de blocos `try...catch` para garantir a integridade das operações.
- **LocalStorage:** Gerenciamento de strings JSON para armazenamento local.
